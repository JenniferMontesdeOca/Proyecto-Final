Tareas del Curso

  • Tarea 1:

    Crear una aplicación simple de lista de tareas que permita al usuario agregar tareas, marcarlas como completadas y eliminarlas. Se debe utilizar una estructura de datos de Lista Enlazada. (Si es doblemente enlazada puntos extras)

    Características:

    • Agregar Tareas: El usuario puede agregar nuevas tareas ingresando un texto en un campo de entrada y presionando un botón para agregarla a la lista.
    • Marcar como Completada: Cada tarea en la lista debe tener una casilla de verificación que permita al usuario marcar la tarea como completada.
    • Eliminar Tareas: El usuario puede eliminar una tarea de la lista haciendo clic en un botón de eliminar junto a la tarea.
    • Persistencia de Datos: Las tareas agregadas por el usuario deben persistir incluso después de que la página se recargue. Puedes utilizar el almacenamiento local del navegador para lograr esto.

    Tecnologías Utilizadas:

    • HTML, CSS para la estructura y estilo de la interfaz de usuario.
    • JavaScript para la lógica de la aplicación y la manipulación de la lista de tareas.
    • Almacenamiento local (Caché) del navegador para persistir los datos de las tareas.

    Utilizar un repositorio de Github y subirla a Github Pages. Adjuntar el Link del repositorio y del Github pages.

  • Tarea 2:

    Con base de lo aprendido en clase, desarrolle el siguiente programa:

    1. Realizar una función que genere 10,000 números aleatorios Enteros y guardarlos en un Array. Números entre 0 y 10000.
    2. Utilizar el array anteriormente para comparar los diferentes algoritmos de ordenamiento en JavaScript y determinar cuál es el método más rápido.
    3. Mostrar al final el resultado de todos los algoritmos y cuál es el veredicto.
    4. Usar los métodos Buble Sort, Selection Sort, Insertion Sort, Merge Sort y Quick Sort.
